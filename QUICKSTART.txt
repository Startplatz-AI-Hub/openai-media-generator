SCHRITT 2️⃣ - CODE HOCHLADEN (1 Minute)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Öffne Terminal/PowerShell in DIESEM Ordner und führe aus:

    cd openai-media-generator-github-pages
    
    git init
    
    git add .
    
    git commit -m "Initial commit"
    
    git branch -M main
    
    git remote add origin https://github.com/DEIN-USERNAME/openai-media-generator.git
    
    git push -u origin main

⚠️ Ersetze "DEIN-USERNAME" mit deinem GitHub Username!

SCHRITT 3️⃣ - GITHUB PAGES AKTIVIEREN (1 Minute)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Gehe zu deinem Repository auf GitHub
2. Klicke auf [Settings] (oben)
3. Linkes Menü → [Pages]
4. Source: "Deploy from a branch"
5. Branch: "main" → Ordner: "/ (root)" → [Save]
6. ⏳ Warte 1-2 Minuten...
7. 🎉 Refresh die Seite - deine URL erscheint!

🆘 PROBLEME?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "404 Not Found"
   → Warte 2-3 Minuten nach dem ersten Deploy
   → Check Settings → Pages → Branch ist "main" / "root"

❌ Git Fehler
   → Stelle sicher, dass du im richtigen Ordner bist
   → Check: git config --global user.name "Dein Name"
   → Check: git config --global user.email "deine@email.com"

❌ Push rejected
   → Hast du die richtige Repository URL?
   → Bist du bei GitHub eingeloggt?
   → Versuche: git push -f origin main (Vorsicht bei bestehenden Repos!)
